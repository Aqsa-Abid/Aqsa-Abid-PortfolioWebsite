Project 1

Discription

This project is a library management system that allows users to search for books, check out books, and return books. The system is implemented in C++ and uses pointers. The project includes the following features:

  • A user interface that allows users to search for books by title or ID.
  • A system for inserting Books
  • A system for issuing books to students
  • A system for returning books.
  • A system for tracking the status of books (e.g., checked out, returned, overdue).
The project was developed using the following technologies:
  • C++
  • Pointers
  • Object-oriented programming
The project was developed over a period of two months. I am proud of the work I did on this project. I learned a lot about C++, pointers, and object-oriented programming.
